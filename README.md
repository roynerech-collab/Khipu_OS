# Khipu OS

Bot personal de Telegram para seguimiento financiero.

## Configuracion local

Este proyecto usa credenciales locales de Telegram. No subas tokens reales a GitHub.

Variables necesarias:

TELEGRAM_TOKEN
MI_TELEGRAM_ID

## Servidor

Para actualizar el codigo en la PC servidor:

ACTUALIZAR.bat

Para iniciar el bot:

LANZADOR.bat
